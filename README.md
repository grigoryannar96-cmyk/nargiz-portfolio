# Nargiz Grigoryan — Portfolio

Personal portfolio website for **Nargiz Grigoryan**, Graphic & Brand Designer based in Yerevan, Armenia.

## About

Dark editorial single-page portfolio showcasing 6+ years of design work across brand identity, print, packaging, social media, and AI content creation.

## Sections

- **Work** — Featured client projects and presentations
- **SMM** — Social media design for Instagram and digital platforms
- **Print** — Editorial, flyer, and print design
- **Branding** — Logo and brand identity systems
- **Packaging** — Product label and packaging design
- **AI Content** — AI-generated character design (Artuyt) and AI photo editing
- **About** — Bio and skills
- **Experience** — Work history

## Tech

- Vanilla HTML / CSS / JavaScript — no frameworks
- Dark theme with CSS custom properties
- Responsive layout
- Autoplay video reels with engagement stats
- Image lightbox viewers per section

## Versions

| File | Purpose |
|------|---------|
| `index.html` | Main portfolio (contact section) |
| `index-upwork.html` | Upwork version — client testimonials + Hire Me CTA |

## Live

Served as a static site via any HTTP server:

```bash
python3 -m http.server 8000
```
